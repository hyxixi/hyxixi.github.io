Axwoman6.0安装（.NET 4.0配置）

Axwoman6.0安装（.NET 4.0配置）

date: 2016-08-01 08:49:54

题记

axwoman必须基于GIS平台，最新版本为axwoman6.0，基于ArcGIS10，另外axwoman1.0基于ArcView3.x，axwoman2.0是JAVA版，axwoman3.0也是基于ArcView，axwoman4.0基于ArcGIS9.2, axwoman5.0基于ArcGIS9.2。这里只谈Axwoman6.0的安装。

正文

　　首先，安装Axwoman6.0之前将Microsoft .net framework version 4.0和ArcGIS 10.0安装好。如果有人需要安装包，建议直接加一个空间句法的交流群。

　　这里简要说一下arcgis的安装步骤：

  • 先安装许可管理器（LicenseManager），不要改任何东西，安好会弹出一个窗口，点击“停止”，关闭许可管理器。
  • 再安装Arcgis Desktop（包括汉化包），不要改任何东西，关闭ArcGIS Administrator窗口，暂不设置。
  • 接着替换service.txt文件，打开KeyGen.exe，设置hostID（自己的计算机名）和Version（Arcgis版本），点击ALL，将所有内容复制下来，替换原来的service.txt文件内容，再用新的service.txt文件覆盖“C:\Program Files (x86)\ArcGIS\License10.0\bin”目录下的service.txt文件。
  • 最后，打开License Server Administrator窗口，点击“运行”，再打开ArcGIS Administrator窗口，选择“ArcInfo（浮动使用）”，许可管理器输入“localhost”，点击“确定”以后，可以看到左边的树结构中出现自己的计算机名，这样就安装结束了。

　　安装Axwoman6.0，只把Just me改为Everyone就好，然后打开Arcmap的扩展模块，可以看到AxialGen2.0和Axwoman6.0。这里注意，如果你在安装时改动了arcgis或者Axwoman的地址，可能就看不到这两项扩展功能。如果都没问题，就是.NET版本不对了。

　　Microsoft .net framework version 4.0的安装也很简单，也是一路next就好。但是这个程序是电脑程序默认安装的，所以在下载安装包之前，先检查电脑的.NET的版本吧。这里有几种方法，是我试过的：

  1. 在文件管理器的地址栏输入“C:\Windows\Microsoft.NET\Framework”或者“C:\Windows\Microsoft.NET\Framework64”，最大版本就是电脑现有的版本。
  2. 打开系统自带的IE浏览器，在地址栏输入“javascript:alert(navigator.userAgent);”，在弹出窗口处标明了系统安装的 .NET Framework 版本信息。
  3. 在“C:\Windows”目录下找到regedit.exe程序，找到“HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\Client”的Version一栏，就是电脑现有的.NET Framework版本。
  4. 打开控制面板中的“程序与功能”，在左上角的“启动或关闭Windows功能”中，有 .NET Framework的版本信息。

　　一般Win7默认安装 .NET Framework 3.5，Win10默认安装 .NET Framework 4.6。版本比4.0低，直接安装4.0就行；版本比4.0高，在“程序与功能”左上角的“启动或关闭Windows功能”中关闭高的版本，在再安装4.0 。如果安装不了，可以试试修改注册表信息。
