修改注册表

date: 2016-07-31 20:50:33

首先，在“C:\Windows”目录下找到regedit.exe程序，运行。

1、单击“开始”，最上方是你的用户名，例如我的是“mf”。

2、在注册表中，定位到任意你想要修改权限的项，右键单击它，选择权限。

3、选择“高级”。

4、看所有者一栏，如果是你计算机名加用户名，就直接做第8步，否则单击“更改”。

5、单击“高级”，下一步。

6、在“查找位置”里输入你的计算机名，单击“立即查找”，找到你的用户名，选中，单击“确定”。

7、回到“高级安全设置”窗口，如果有替换子容器和对象的所有者的复选框，勾上。

8、回到最初的权限窗口，选择用户名Administrators，设置权限，如图，单击“确定”。

9、之后，你就可以修改注册表了。

Axwoman6.0安装（.NET 4.0配置）

date: 2016-07-31 18:49:54

题记

axwoman必须基于GIS平台，最新版本为axwoman6.0，基于ArcGIS10，另外axwoman1.0基于ArcView3.x，axwoman2.0是JAVA版，axwoman3.0也是基于ArcView，axwoman4.0基于ArcGIS9.2, axwoman5.0基于ArcGIS9.2。这里只谈Axwoman6.0的安装。

正文

　　首先，安装Axwoman6.0之前将Microsoft .net framework version 4.0和ArcGIS 10.0安装好。如果有人需要安装包，建议直接加一个空间句法的交流群。

　　这里简要说一下arcgis的安装步骤：

  • 先安装许可管理器（LicenseManager），不要改任何东西，安好会弹出一个窗口，点击“停止”，关闭许可管理器。
  • 再安装Arcgis Desktop（包括汉化包），不要改任何东西，关闭ArcGIS Administrator窗口，暂不设置。
  • 接着替换service.txt文件，打开KeyGen.exe，设置hostID（自己的计算机名）和Version（Arcgis版本），点击ALL，将所有内容复制下来，替换原来的service.txt文件内容，再用新的service.txt文件覆盖“C:\Program Files (x86)\ArcGIS\License10.0\bin”目录下的service.txt文件。
  • 最后，打开License Server Administrator窗口，点击“运行”，再打开ArcGIS Administrator窗口，选择“ArcInfo（浮动使用）”，许可管理器输入“localhost”，点击“确定”以后，可以看到左边的树结构中出现自己的计算机名，这样就安装结束了。

　　安装Axwoman6.0，只把Just me改为Everyone就好，然后打开Arcmap的扩展模块，可以看到AxialGen2.0和Axwoman6.0。这里注意，如果你在安装时改动了arcgis或者Axwoman的地址，可能就看不到这两项扩展功能。如果都没问题，就是.NET版本不对了。

　　Microsoft .net framework version 4.0的安装也很简单，也是一路next就好。但是这个程序是电脑程序默认安装的，所以在下载安装包之前，先检查电脑的.NET的版本吧。这里有几种方法，是我试过的：

  1. 在文件管理器的地址栏输入“C:\Windows\Microsoft.NET\Framework”或者“C:\Windows\Microsoft.NET\Framework64”，最大版本就是电脑现有的版本。
  2. 打开系统自带的IE浏览器，在地址栏输入“javascript:alert(navigator.userAgent);”，在弹出窗口处标明了系统安装的 .NET Framework 版本信息。
  3. 在“C:\Windows”目录下找到regedit.exe程序，找到“HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\Client”的Version一栏，就是电脑现有的.NET Framework版本。
  4. 打开控制面板中的“程序与功能”，在左上角的“启动或关闭Windows功能”中，有 .NET Framework的版本信息。

　　一般Win7默认安装 .NET Framework 3.5，Win10默认安装 .NET Framework 4.6。版本比4.0低，直接安装4.0就行；版本比4.0高，在“程序与功能”左上角的“启动或关闭Windows功能”中关闭高的版本，在再安装4.0 。如果安装不了，可以试试修改注册表信息。

办理手机卡和手机套餐注意事项

date: 2016-07-23 16:12:30

题记

现在，大多数学生都在初中甚至更早的时候就有了私人手机，但有些学生是高考后才买，我就是，当时的盲目购买，让我之后的2年后悔不已。如今，对于办理手机卡和手机套餐，我终于有了一个比较清晰的认识，写下此文，希望需要的人能看到。

正文

1.如果你像我一样，高考完准备买手机，建议你最好到学校再买，开学季各大高校的营业厅，推出很多优惠的校园套餐，现在，联通推出校内流量包10元1.5G，但限3G用户，还有其他限制，不清楚。

2.如果你在校办卡，当地营业厅是不支持异地销卡的，你可以打电话给客服，让客服帮你停机，90天后自动销号。

3.无论是电话在线办理业务，还是当地营业厅办理业务，均在48h内的某一时刻生效，一旦生效，不能取消，只能在下个月正式生效后再处理。

4.当地营业厅办理业务必须要有效身份证，过期身份证不能办理业务，如果对方没发现，那么恭喜你，蒙混过关，否则，你只能电话办理业务。

5.网速与你的手机配置有直接关系，如果你是4G的卡，2G的手机，你的手机网速就是2G，同样，你是4G的手机，2G的卡，你的手机网速最快就是4G。

6.如果你是学生，建议你办理校园手机卡，校园手机卡在校内非常优惠，偶尔出省，可以订购全国流量，如果你担心网速慢，可以卖一个4G手机。

7.2G用户、3G用户可以转4G用户，4G用户不能转回3G用户，遇到这种情况，你只能销卡了。

8.买手机的时候都要冲100元话费，销卡的话必须满3个月，100元的话费用完，而且没有任何还未正式生效的业务，没有每月返现的业务，才能销卡。

地址分词快速分辨

date: 2016-07-23 12:16:24

　　最近做了一些分辨地址分词的工作，总结了一些小技巧，可以快速提高大家的工作速度。

1.对地址分词进行筛选

　　做地址分词工作一般是先判断对错，再判断错误类型，判断对错时要对地址分词进行筛选，判断错误类型时要对任意一列的颜色进行筛选（我采用标红标错）。

2.快速分辨地址中有没有括号

　　无论地址中有没有括号，地址分词中都不该有括号出现，这种情况不用想太多，直接标错；若括号中的地址丢失，也标错。

3.对比地址与地址分词的尾部

　　尾部丢失是分词时比较常出现的错误了，一般是在地址尾部，特别是，尾部有顿号链接的多个TAB。

4.分辨地址中有没有“%组”或者“%楼”

　　在地址分词中经常把“%组”标为TAB，“%楼”标成BUL。

5.分辨地址中有没有“%超市”

　　在地址分词中经常把“%超市”标为CIT。

6.分辨地址分词中有没有CRS

　　大多数CRS前没有“交叉口”都是错的，当然这是指分词对了，出现丢失字段现象；如果地址中没有“交叉口”三个字，也没有“%路%路”，一般就是分词分错了。

7.分辨地址分词中有没有COM

　　地址分词的稳定结构是PNA+TAB，如果是COM+TAB,直接标错；有时候，COM前面只有2个字，后面跟着的PNA是镇开头,也是错的；而COM后面没跟镇，地址中却是“%镇”，则属于分词丢失错误。

8.分辨地址分词中有没有CAT

　　如果CAT前只分出一个字，例如“镇”“村”，都是错的；如果是COM+CAT的结构，“苏巷镇/COM 农贸市场/CAT”是对的，“苏巷/COM 农贸市场/CAT”是错的，应该是“苏巷农贸市场/PNA”。

9.分辨地址分词中有没有太长字段

　　偶尔COM+PNA会一起被分为PNA，PNA+PNA合在一起分成一个PNA，也是不可以的。

10.分辨地址分词中有没有STR

　　如果STR前面只有2个字，后面跟着的地址分词是路开头，那么分词一般是错的，如果不是，STR不一定就错的，STR+TAB是稳定结构，但是偶尔后面会跟PNA。

11.分辨地址分词中有没有FIX

　　FIX在分词时，可能出现这样的情况：“南北楼/PNA”分成“南/FIX 北/BUL”。

尾记

　　这些是比较容易辨别的错误，如果这些错误都没有，还是要对地址与地址分词作最基本的核对，当然，如果你看此文不太懂，请先看“地名地址分词基础规则.docx”。
